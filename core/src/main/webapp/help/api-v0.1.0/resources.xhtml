Resources

A resource is an entity that is managed by Hour Log. For example, a facility, a vault, a log entry, a shift, a beam etc. The #{msgs.title} API provides access to the resources or entities that Hour Log manages. These entities and their attributes are described below. Note that the names of attributes are important as they are used in the various representations of these resources (see Resource Representations Section below).
